2020-08-16

什么是装饰器

• java 中称之为注解
• javascript 中装饰器的作用更加强大
• 装饰器是 方法 或者 类 的重定义
• 实现原理是 Object.defineProperties
• 装饰器 在运行前 并不是真实的可用代码 而是对代码的描述与修改
• 装饰器 不是 runtime
